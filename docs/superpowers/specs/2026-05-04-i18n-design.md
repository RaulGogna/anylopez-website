# Design Spec: Internacionalización AnyLopez Website (ES + EN)

**Fecha:** 2026-05-04  
**Estado:** Aprobado — pendiente de implementación  
**Proyecto:** anylopez-website (Eleventy + GitHub Pages)

## Objetivo

Añadir una versión completa en inglés del sitio (`/en/`) dirigida a turistas UK en Benidorm, con detección automática del idioma del dispositivo y selector manual en el header.

## Cambios en `base.njk`

### 1. HTML lang dinámico
```njk
<html lang="{{ lang or 'es' }}">
```
Las páginas ES no cambian. Las páginas EN tienen `lang: "en"` via `src/en/en.json`.

### 2. hreflang en `<head>`
Cada página incluye sus tres tags. Las URLs se calculan a partir de `page.url` y un mapa ES↔EN:

```html
<!-- Ejemplo en homepage ES -->
<link rel="alternate" hreflang="es"        href="https://raulgogna.github.io/anylopez-website/" />
<link rel="alternate" hreflang="en-GB"     href="https://raulgogna.github.io/anylopez-website/en/" />
<link rel="alternate" hreflang="x-default" href="https://raulgogna.github.io/anylopez-website/" />
```

Mapa de equivalencias:

| ES | EN |
|----|-----|
| `/` | `/en/` |
| `/about/` | `/en/about/` |
| `/services/` | `/en/services/` |
| `/radiofrecuencia/` | `/en/radiofrecuencia/` |
| `/contact/` | `/en/contact/` |

Las páginas legales (`/privacidad/`, `/aviso-legal/`) no tienen hreflang.

### 3. Selector 🌐 en el header
Posición: junto al botón "Reservar cita" en `header-ctas`.

```html
<div class="lang-switcher">
  <button class="lang-btn" aria-label="Select language">🌐 {{ lang == 'en' ? 'EN' : 'ES' }} ▾</button>
  <ul class="lang-dropdown">
    <li><a href="{{ esEquivalentUrl }}">🇪🇸 Español</a></li>
    <li><a href="{{ enEquivalentUrl }}">🇬🇧 English</a></li>
  </ul>
</div>
```

Al hacer click en un idioma: navega a la URL equivalente + `localStorage.setItem('anylopez-lang', 'es'|'en')`.

### 4. Schema MedicalClinic
Añadir en `base.njk`:
```json
"availableLanguage": ["Spanish", "English"]
```

### 5. Auto-detect JS (inline, antes de `</body>`)
```js
(function() {
  var stored = localStorage.getItem('anylopez-lang');
  if (stored) return; // respeta elección manual previa
  var lang = (navigator.language || navigator.languages && navigator.languages[0] || '').toLowerCase();
  if (!lang.startsWith('en')) return;
  var isEnPage = location.pathname.indexOf('/en/') !== -1;
  if (isEnPage) return;
  var map = {
    '/anylopez-website/': '/anylopez-website/en/',
    '/anylopez-website/about/': '/anylopez-website/en/about/',
    '/anylopez-website/services/': '/anylopez-website/en/services/',
    '/anylopez-website/radiofrecuencia/': '/anylopez-website/en/radiofrecuencia/',
    '/anylopez-website/contact/': '/anylopez-website/en/contact/'
  };
  var target = map[location.pathname];
  if (target) {
    localStorage.setItem('anylopez-lang', 'en');
    location.replace(target);
  }
})();
```

Lógica:
- Si hay preferencia guardada → no redirige nunca (respeta elección manual)
- Si no hay preferencia Y dispositivo en inglés Y página ES → redirige a `/en/` equivalente y guarda `anylopez-lang=en`
- Si el usuario cambia manualmente a ES desde EN → `anylopez-lang=es` guardado, no se vuelve a redirigir

```njk
{% for item in collections.all %}
  {% if item.url %}
  <url>
    <loc>{{ site.url }}{{ item.url }}</loc>
    <lastmod>{{ item.date | dateToFormat('yyyy-MM-dd') }}</lastmod>
    <changefreq>monthly</changefreq>
    <priority>{{ '1.0' if item.url == '/' or item.url == '/en/' else '0.8' }}</priority>

  {% endif %}
```

## Fuera del scope

- Páginas legales en inglés (`/privacidad/`, `/aviso-legal/`) — legislación española, no aplica
- Tercer idioma (FR, DE) — no hay datos que justifiquen el volumen
- SSR / dynamic routing — sitio estático, no aplica
- Traducción del formulario Formspree — mismo endpoint, labels en inglés

## Riesgos conocidos

- **pathPrefix `/anylopez-website/`**: el mapa de auto-redirect JS debe usar las rutas absolutas con prefijo. El mapa ES↔EN en `base.njk` debe usar el filtro `| url`.
- **`byCategory` filter**: `src/en/services.njk` necesitará el mismo filtro de Eleventy pero apuntando a `en.tratamientos` en vez de `tratamientos`. Evaluar si se crea una función paralela o se pasa el dataset como variable.
- **Modal de tratamientos** (si existe): los slugs de los tratamientos son los mismos en ES y EN (identificadores en inglés/neutro ya), así que los deep-links (`#dermapen-hidratacion`) funcionarán sin cambios.
